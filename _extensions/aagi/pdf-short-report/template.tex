  Report text goes here. This template is designed for smaller summary/interim
  reports, typically less than 10 pages or so. The AAGI logo and partners footer
  appear only on this first page to maximise space for the body text. Feel free
  to remove the Table of Contents if the report is short enough to not need one.

  Use the AAGI theme colours for emphasis if you like (\verb|\color{aagiteal}|
  for {\color{aagiteal}AAGI teal}, \verb|\color{aagigreen}| for
  {\color{aagigreen}AAGI green}, and \verb|\color{aagiblack}| for
  {\color{aagiblack}AAGI black}).

  \subsection{This is a subsection heading}

  Example R code (using the basic \texttt{lstlisting} environment):

\begin{lstlisting}
plot(pressure)
\end{lstlisting}

  Use \verb|[@blah]| for parenthetical citations: [@blah]. Use \verb|@blah|
  for text citations, e.g.\ @blah. References are entered in BibTeX format
  in your \texttt{.bib} file and declared in the \texttt{bibliography:} field
  of your \texttt{.qmd} YAML front matter.

  A simple AAGI-themed table could be done as in \autoref{tab:aagi}.
  \begin{table}[h]
    \caption{An AAGI-themed table.}\label{tab:aagi}
    \centering
    \begin{tabular}{| l | r r |}
    \rowcolor{aagiteal} {\color{white}Table Heading} & %
    {\color{white}Next Heading} & {\color{white}Another Heading} \\
    \hline
    blah & 2 & 3 \\
    \hline
    \end{tabular}
  \end{table}

  \subsubsection{This is a subsubsection heading}

  And so on, etc.

  Use \verb|\begin{note} ... \end{note}| for emphatic notes throughout
  the report if you like. You can use \verb|\faIcon{...}| to add
  FontAwesome icons for decoration, as shown below.

  \begin{note}
    {\sbf\color{aagiteal!75!black} \faIcon{exclamation-triangle} NOTE}\quad
    This is an example of a note.
  \end{note}

      \end{axis}
    \end{tikzpicture}

  \end{figure}
